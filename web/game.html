NET Game



    

        

            

                Here is the NET game.

                In this game, you need to connect all the pieces together. When two pieces are connected, their colors change from pink to blue.
            

        
            

                If the little light in the top-right corner of the game is on, you are in wrapping mode.
            

        
            

                The HELP menu is there to guide you through the different available commands.
            

        
            

                A LEFT CLICK rotates a piece clockwise, and a RIGHT CLICK rotates it counterclockwise.
            

        

        

        
        
    

        
Help Menu

        

        'R': Create a random game 

        'Z': Undo the last move 

        'Y': Redo the last move 

        'K': Solve the game, up to 10 seconds

        'M': Shuffle the game

        'S': Save the game

        'L': Load an existing game
